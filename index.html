如何在Eclipse中创建J2EE项目？（非Maven项目）

Eclipse中的J2EE项目就是Dynamic Web Project项目。接着把Default output folder修改为WebRoot\WEB-INF\classes（该目录结构遵循MyEclipse中的Java Web Project项目结构），这时编译后的class文件就会放在该目录下。

J2EE项目的jar包放在哪？（非Maven项目）

在WebRoot\WEB-INF目录下新建lib文件夹，接着把jar包都拷贝进去，然后在项目右键–>properties–>Java Build Path中引用lib下的jar包，使得jar包可以参与项目编译。

项目源文件如何部署到中间件中？

在项目右键–>Properties–>Deployment Assembly中查看。tomcat启动时可能会报找不到Java类，这时便可以到这里去查看哪些java源文件、jar包发布到了中间件中。

找不到Java类的原因主要有：

  1. java源文件、jar包未发布到中间件中。
  2. jar包冲突，典型的案例就是同时发布jetty和tomcat的jar包后发生冲突

如何拉取并部署一个基于Maven的J2EE项目

第一步：先用git工具把整个项目clone到你的工作空间下。

第二步：打开cmd并cd到项目目录，执行下列命令。该命令会做这些事：

  1. 把项目转变为eclipse项目，生成.settings文件夹、.classpath文件、.project文件。
  2. 从maven中央库下载jar包
  3. 编译项目

第三步：导入项目至eclipse中：Eclipse–>File–>Import–>General–>Existing Projects into Workspace–>选择项目。
